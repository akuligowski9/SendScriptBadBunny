# 🐰 SendScriptBadBunny

Envía la letra completa de **DeBÍ TiRAR MáS FOToS** de Bad Bunny, línea por línea, por WhatsApp.

Porque a veces la persona que se merece leer esto tarda demasiado en escuchar el álbum.

---

## ¿Cómo funciona? / How to use

1. Abre [WhatsApp Web](https://web.whatsapp.com) en Chrome o Firefox / Open [WhatsApp Web](https://web.whatsapp.com) in Chrome or Firefox
2. Abre el chat de tu víctima favorita / Open the chat of your favorite victim
3. Abre la consola del navegador: `F12` (Windows) o `Cmd + Option + J` (Mac) / Open the browser console: `F12` (Windows) or `Cmd + Option + J` (Mac)
4. Si Chrome dice **"allow pasting"**, escribe eso y presiona Enter / If Chrome says **"allow pasting"**, type that and press Enter
5. Copia todo el contenido del archivo `.js` que quieras usar / Copy the entire contents of the `.js` file you want to use
6. Pégalo en la consola y presiona Enter / Paste it into the console and press Enter
7. Observa / Watch
